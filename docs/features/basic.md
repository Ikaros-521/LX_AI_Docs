# 基础功能

## 统一配置

统一配置是系统的全局设置，用于管理系统级别的参数和选项。

### 配置项说明

- **系统设置**
  - 系统名称：显示在界面标题和日志中的系统名称
  - 调试模式：是否启用调试日志
  - 日志级别：系统日志的详细程度

- **任务配置**
  - 任务队列大小：系统可以同时处理的最大任务数
  - 任务超时时间：单个任务的最大执行时间
  - 重试策略：任务失败后的重试机制

### 使用说明

1. 访问统一配置页面
2. 修改相关参数
3. 保存配置（部分配置可能需要重启服务生效）

## 配置管理

配置管理用于管理不同场景下的配置组合，支持多配置切换和导入导出。

### 配置列表

在配置列表中，您可以：

1. 创建新配置
   - 点击"新建配置"按钮
   - 填写配置名称和说明
   - 选择平台和功能模块
   - 设置具体参数

2. 管理现有配置
   - 编辑：修改配置内容
   - 删除：移除不需要的配置


## 功能管理

### 1. 消息优先级

消息优先级功能用于管理不同类型消息的处理顺序。

#### 优先级设置
- 高优先级：礼物、付费内容
- 中优先级：普通互动、关注
- 低优先级：系统消息、闲时任务

### 2. 消息过滤

消息过滤功能帮助过滤无效或不需要的消息。

#### 过滤规则
- 关键词过滤
- 用户等级过滤
- 消息长度过滤
- 频率限制

### 3. 违禁词设置

违禁词功能用于自动过滤不当内容。

#### 功能特点
- 分级过滤
- 自动替换

#### 使用方法
1. 添加违禁词
2. 设置处理方式
3. 配置替换规则

### 4. 弹幕&回复模板

用于管理自动回复的模板系统。

#### 模板变量
- `{username}`: 用户名
- `{comment}`: 弹幕内容

#### 模板示例
```
{username}[说|说道]:{comment}
```

### 5. 答谢功能

自动答谢系统，用于感谢观众的互动。

#### 支持类型
- 礼物答谢
- 关注答谢
- 入场答谢

#### 配置项
- 触发条件
- 答谢文案
- 延迟设置
- 合并规则

### 6. 闲时任务

闲时任务用于在直播间互动较少时维持活跃度。

#### 运行机制
1. 待播放音频队列更新闲时
2. 待合成消息队列更新闲时
3. 直播间无消息更新闲时

#### 配置说明
- 触发条件
- 执行间隔
- 任务内容
- 优先级设置

### 7. 定时任务

定时任务用于按计划执行特定操作。

#### 任务类型
- 固定时间任务
- 周期性任务
- 条件触发任务

#### 配置方法
1. 设置执行时间
2. 配置任务内容
3. 设置重复规则

### 8. 本地问答

本地问答系统。

#### 功能特点
- 智能匹配

#### 使用步骤
1. 导入问答数据
2. 配置匹配规则
3. 测试响应效果

## 最佳实践

1. 配置管理
   - 定期备份配置
   - 使用模板快速部署
   - 及时更新配置

2. 性能优化
   - 合理设置优先级
   - 适当的过滤规则
   - 控制任务频率

3. 内容管理
   - 定期更新违禁词库
   - 优化回复模板
   - 维护问答库

## 常见问题

1. 配置不生效
   - 检查配置格式
   - 确认保存成功
   - 可能需要重启服务

2. 任务执行异常
   - 查看错误日志
   - 检查触发条件
   - 验证配置正确性

3. 性能问题
   - 优化过滤规则
   - 调整任务频率
   - 清理历史数据
